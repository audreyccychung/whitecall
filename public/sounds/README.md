# Sound Files

This directory should contain the following sound files for WhiteCall:

- `heart-sent.mp3` - Played when user sends a heart
- `heart-received.mp3` - Played when user receives a heart
- `success.mp3` - Generic success sound

## Placeholder Sounds

For development, you can:
1. Use free sound effects from sites like freesound.org or zapsplat.com
2. Create simple sounds using tools like Audacity
3. Generate sounds with online tools

The sounds should be:
- Short (< 1 second)
- Soft and pleasant
- Professional but friendly
- Small file size (< 50KB each)

## Recommended Sounds

- **heart-sent**: Gentle "whoosh" or soft chime
- **heart-received**: Warm "ding" or soft sparkle
- **success**: Light "pop" or cheerful tone

# Changelog

All notable changes to Shift Calendar will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Planned for MVP (v0.1.0)
- Authentication system (email, Google, Apple)
- Monthly calendar view
- Shift template creation and management
- Quick-tap shift addition interface
- Multi-day shift support (8-34 hours)
- Group creation and management
- Group calendar view (weekly blocks with member rows)
- Smart availability finder
- Basic statistics (hours worked, call count)
- Mobile-responsive design

### Planned for v0.2.0 (Post-MVP)
- Calendar sync with Google Calendar
- Calendar sync with Apple Calendar
- iCal feed generation for family sharing
- Stripe payment integration
- Premium tier features activation
- User onboarding flow
